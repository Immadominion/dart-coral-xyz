# Examples

Standalone Dart examples demonstrating `coral_xyz` usage.

## Files

- **basic_usage.dart** — Core components: connection setup, keypair handling, IDL loading, program creation
- **counter_basic.dart** — Simple counter program interaction (mirrors the Anchor "basic-1" TypeScript tutorial)

## Running

```bash
dart run example/basic_usage.dart
dart run example/counter_basic.dart
```

Both examples run without a local Solana validator. Network calls will fail gracefully in demo mode.
